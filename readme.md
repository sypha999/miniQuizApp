# 🧠 Quiz App (Mini Quiz Application)

A simple, interactive quiz application built with JavaScript that allows users to answer questions, receive instant feedback, and track their performance.

🌐 **Live Demo:** https://qui3ify.netlify.app/

---

## 📌 Overview

This project is a lightweight quiz platform designed to demonstrate core frontend concepts such as:

- API consumption (fetching quiz questions)
- DOM manipulation
- State management in vanilla JavaScript
- User interaction handling
- Score tracking and feedback display

It is deployed on Netlify for easy access and continuous delivery.

---

## 🚀 Features

- 🎯 Fetches random quiz questions dynamically
- ✅ Displays multiple-choice answers
- ⚡ Instant feedback on selected answers
- 📊 Tracks user score
- 🔄 Ability to move through questions
- 🌐 Fully deployed and accessible online
- 💡 Clean and minimal UI

## ⚙️ How It Works

1. The app loads and fetches quiz questions from an external API
2. A question is displayed with multiple answer options
3. The user selects an answer
4. The app:
  - Checks correctness
  - Updates the score
  - Provides feedback
5. The next question is displayed until the quiz ends

## 📦 Installation & Setup

To run the project locally:

### 1. Clone the repository

`git clone https://github.com/sypha999/miniQuizApp.git
cd miniQuizApp`

### 2. Install dependencies (if any)
`npm install`

### 3. Run the project
Since this is a frontend app, you can simply open:
index.html
OR use a local server (recommended):
npx server
